# Blush And Bloom – Beauty Ecommerce Website

Blush And Bloom is a responsive frontend beauty ecommerce website developed using **HTML, CSS, and JavaScript**. The project showcases beauty and fashion products including makeup, skincare, accessories, and jewellery with an interactive shopping cart and checkout system. 

---

# Features

* Responsive and aesthetic UI design
* Makeup product section
* Skincare product section
* Accessories collection
* Jewellery collection
* Add to Cart functionality
* Checkout page
* Order confirmation page
* LocalStorage-based cart system
* Interactive buttons and navigation

---

# Technologies Used

* HTML5
* CSS3
* JavaScript
* LocalStorage API

# Main Functionalities

## Shopping Cart

Users can add products to the cart using JavaScript and LocalStorage.

## Checkout System

Users can enter:

* Name
* Email
* Address

before confirming their order.

## Order Confirmation

Displays successful order confirmation after checkout.
